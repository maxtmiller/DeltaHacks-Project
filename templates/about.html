Features

Our web app leverages advanced technologies to detect and manage skin cancer, offering the following features:

  1. User image uploads or webcam captures analyzed by a TensorFlow model to predict malignancy or benign status with a confidence score
  2. SQL database storage for reviewed images
  3. A “Help” section using the Google Maps API to locate nearby hospitals
  4. A Cohere-powered chatbot that answers skin disease questions
  5. Secure authentication via Google OAuth, ensuring a smooth user experience
